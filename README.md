# Manyeleti_Football_Association_League
The Web page to keep track of Manyeleti Football Association League progress
⚽ Manyeleti Football Association League System
A simple and interactive football league management web application built using pure HTML, CSS, and JavaScript in a single file.
This system allows users to manage fixtures, submit match results, and automatically generate an updated league table.

📌 Overview
The Manyeleti Football Association League System is designed to:

Track football teams and their performance
Automatically calculate league standings
Manage fixtures and results in real-time
Provide a simple interface for league updates

This project is ideal for local leagues, school competitions, or community tournaments.

🚀 Features

✅ Automatic fixture generation (all teams play each other)
✅ Real-time league table updates
✅ Points system (Win = 3, Draw = 1, Loss = 0)
✅ Goal tracking:

Goals For (GF)
Goals Against (GA)
Goal Difference (GD)


✅ Input match scores easily
✅ Reset league functionality
✅ Responsive and clean design
✅ Runs entirely in one HTML file (no setup required)


🛠️ Technologies Used

HTML5 – Structure
CSS3 – Styling
JavaScript (Vanilla JS) – Logic & interactivity
